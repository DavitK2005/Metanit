# Упражнения по C++ (Metanit)

Этот репозиторий содержит мои решения задач по C++, взятых с сайта Metanit.

## О проекте

Я использую этот репозиторий для практики и закрепления знаний по C++.
Задачи охватывают основные темы языка:

- базовый синтаксис C++
- указатели и ссылки
- классы и объекты
- перегрузка операторов
- стандартная библиотека (STL)
- контейнеры (vector, map, set, queue и др.)
- алгоритмы
- работа со строками
- ООП и принципы проектирования

## Цель

- Закрепить теоретические знания C++
- Развить навыки решения задач
- Улучшить понимание STL и ООП
- Подготовка к собеседованиям и практическим задачам

## Как запустить

Пример компиляции:

```bash
g++ main.cpp -o app
./app
```
